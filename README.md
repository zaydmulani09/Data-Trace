# 🚀 DataTrace

DataTrace is a lightweight data tracking + analysis web app built for quick insights and clean visualization. It was created as a hackathon project with a focus on simplicity, speed, and real-time interaction.

## 🌐 Live App

[https://datatrace-eta.vercel.app](https://datatrace-eta.vercel.app)

## 🧠 Overview

DataTrace helps users input, track, and analyze data in a simple interface. The goal was to build something that feels fast, minimal, and actually useful without overcomplicating things.

## ⚙️ Tech Stack

* TypeScript
* HTML + CSS
* Vite
* Node.js
* Gemini API (for AI features)

## 📦 Features

* Clean and responsive UI
* Fast local development with Vite
* Modular structure for easy expansion

## 🛠️ Getting Started

### Prerequisites

Make sure you have Node.js installed

### Installation

```bash
npm install
```

### Setup Environment Variables

Create a `.env.local` file and add:

```env
GEMINI_API_KEY=your_api_key_here
```

### Run the App

```bash
npm run dev
```

Then open the local server link in your browser

## 📁 Project Structure

```
src/           # main app code  
index.html     # entry point  
vite.config.ts # vite config  
package.json   # dependencies + scripts  
```
